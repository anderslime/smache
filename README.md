## Tests

To run the tests, run:

`python setup.py test`

### Continous tests for development flow (OS X)

#### Setup

```
brew install terminal-notifier
pip install pytest-watch pytest-flakes pytest-pep8
```

#### Usage

```
./bin/watchtests
```
